# Customer-analytics
Mimecast AI Analytics Engineer Take-Home Assessment

<br> This project analyzes customer acquisition, expansion, and churn across multiple business functions using SQL, AI, and Tableau.

# Task Overview
<br> SQL Task: Built a customer lifecycle data model to track acquisition, expansion, and churn events.
<br> AI Task:  Derived strategic prompts and generate data insights.
<br> Report:   Designed an executive Tableau dashboard and presentation summarizing key insights for Sales, Marketing, and Customer Success.

# Files Included
<br> 01_customer_journey_view.sql – SQL code 
<br> 01_Dimensional Model.pdf – Documentation of the model and joins
<br> 02_AI_Powered_Business_Insights.pdf – Details of the AI-driven analysis task
<br> 03_Customer_Analytics.twbx – Tableau dashboard
<br> 03_Customer Executive Analysis.pptx – Summary presentation

#Tools Used
<br> SQL – Data modeling and transformations
<br> AI  – ChatGPT-5, Grok 3, Open AI Prompt Optimizer - Prompt-based analysis
<br> Tableau – Visualization and dashboard design
